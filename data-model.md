# データモデル定義

---

# Notion登録フィールド

## 必須項目

- タイトル（title）
- 参照URL（url）
- 参加お勧め度（number, 1–5）
- ステータス（select: 候補 / 要確認）

---

# 記載ルール

- 参加お勧め度はONESTRUCTION（建設×BIM×AI）との親和性に基づき1〜5で設定する
- ステータス「候補」: 応募検討に値する
- ステータス「要確認」: 情報不足・期限不明

---

# AI出力JSONスキーマ（固定）

```json
{
  "タイトル": "",
  "参照URL": "",
  "参加お勧め度": 3,
  "ステータス": "候補",
  "is_active": true
}
```

## フィールド説明

| フィールド | 型 | 説明 |
|-----------|-----|------|
| タイトル | string | プログラム名 |
| 参照URL | string | 元ページURL |
| 参加お勧め度 | number (1-5) | ONESTRUCTIONとの親和性スコア |
| ステータス | select | "候補" または "要確認" |
| is_active | boolean | 現在募集中か（false=Notionに登録しない） |

## is_active 判定ルール

- `true`: 現在応募受付中、または判断できない場合
- `false`: 期限切れ・募集終了・終了済みと読み取れる場合（Notionに登録されない）
